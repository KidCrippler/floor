# דו-קרב טריוויה (Duel Trivia Game)

משחק טריוויה דו-קרבי בעברית עם תמיכה מלאה ב-RTL, מותאם לתצוגה על מסך גדול.

## תכונות עיקריות

- ✅ ממשק בעברית מלא עם תמיכה RTL
- ⏱️ שני שעוני ספירה לאחור עצמאיים
- 🎮 שליטה מלאה במקלדת
- 📂 קטגוריות שאלות מרובות
- 🔄 מחזור אוטומטי של שאלות כשנגמרות
- 🎵 מוזיקת רקע וצלילי אפקט
- 📊 מסך ניצחון עם סטטיסטיקות
- 🎨 אנימציות ומשוב ויזואלי

## דרישות מערכת

- דפדפן מודרני (Chrome מומלץ)
- אין צורך בחיבור לאינטרנט (למעט טעינה ראשונית של פונט)

## הוספת קבצי אודיו (אופציונלי)

1. הוסף קבצי MP3 לתיקייה `/sounds/`:
   - `countdown.mp3` - ספירה לאחור מלאה (3, 2, 1, GO!)
   - `correct.mp3` - צליל תשובה נכונה
   - `skip.mp3` - צליל דילוג
   - `gameover.mp3` - סיום משחק
   - `background.mp3` - מוזיקת רקע (תתנגן בלופ)

2. **איפה למצוא צלילים?** ראה `SOUND_EFFECTS_GUIDE.md` למדריך מפורט עם לינקים!

3. המשחק יעבוד גם בלי קבצי אודיו (ייפלט אזהרה בקונסול)

## שימוש במשחק

### מסך הגדרות

1. בחר קטגוריה מהרשימה
2. הזן שמות מתמודדים (ברירת מחדל: "ימין" ו"שמאל")
3. הגדר זמן התחלתי לכל מתמודד (ברירת מחדל: 45 שניות)
4. הגדר עונש דילוג (ברירת מחדל: 3 שניות)
5. בחר מי מתחיל
6. לחץ "התחל משחק"

### שליטת מקלדת במשחק

| מקש | פעולה |
|-----|-------|
| **SPACE** | תשובה נכונה → החלף מתמודד + שאלה הבאה |
| **S** | דילוג → הצג תשובה למשך 3 שניות + שאלה הבאה |
| **ESC** | השהה/המשך משחק |
| **R** | התחל מחדש (עם אישור) |

### מהלך המשחק

1. **במהלך תור רגיל**:
   - השעון של המתמודד הפעיל סופר לאחור
   - לחץ SPACE אם התשובה נכונה → המעבר למתמודד השני
   - לחץ S אם המתמודד מדלג → הצג תשובה למשך 3 שניות (השעון ממשיך לרוץ!)

2. **במהלך דילוג (S)**:
   - התשובה הנכונה מוצגת על המסך
   - השעון ממשיך לספור לאחור במשך 3 שניות מלאות
   - לאחר 3 שניות, התשובה נעלמת והשאלה הבאה מופיעה
   - אותו מתמודד ממשיך (כבר ניכו 3 שניות מהזמן שלו)

3. **אזהרת זמן נמוך**:
   - כאשר נשארות 10 שניות או פחות, השעון הופך לאדום ופועם
   - מתנגן צליל אזהרה (אם קיים)

4. **סיום המשחק**:
   - כאשר שעון אחד מגיע ל-0, המשחק נגמר
   - מוצג מסך ניצחון עם סטטיסטיקות:
     - מספר תשובות נכונות
     - מספר דילוגים
     - זמן שנותר למנצח

## הוספת שאלות

ערוך את הקובץ `js/questionsData.js` והוסף קטגוריות ושאלות חדשות:

```javascript
const QUESTIONS_DATA = {
  "קטגוריה חדשה": [
    {
      "type": "text",
      "content": "שאלה בעברית?",
      "answer": "תשובה בעברית"
    },
    {
      "type": "image",
      "content": "images/my-image.png",
      "answer": "תשובה לתמונה"
    }
  ],
  // ... קטגוריות אחרות
};
```

### הוספת שאלות תמונה

1. שים את קבצי התמונה בתיקייה `/images/`
2. הוסף שאלה עם `"type": "image"` ונתיב התמונה

## תפעול על מסך גדול

1. פתח את המשחק בדפדפן במסך מלא (F11)
2. חבר את המחשב למקרן או טלוויזיה
3. כוונן את גודל התצוגה לפי הצורך (Ctrl + / Ctrl -)

## פתרון בעיות

### השאלות לא נטענות
- ודא שקובץ `questions.json` תקין (בדוק ב-jsonlint.com)
- ודא שאין בעיות קידוד (השתמש ב-UTF-8)

### הפונט לא נראה טוב
- ודא חיבור לאינטרנט לטעינה ראשונית של Google Fonts
- או הורד את הפונט מקומית

### קבצי אודיו לא מתנגנים
- ודא שהקבצים נמצאים בתיקייה `/sounds/`
- ודא שהקבצים בפורמט MP3
- בדפדפנים מסוימים, אודיו אוטומטי נחסם - לחץ על המסך פעם אחת

### המשחק לא מתחיל
- בדוק את הקונסול בדפדפן (F12) לשגיאות
- ודא שהגדרות נשמרו במסך ההגדרות
