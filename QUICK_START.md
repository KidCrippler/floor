# התחלה מהירה - דו-קרב טריוויה

## 🚀 כך מתחילים

### צעד 1: פתח את המשחק
פתח את הקובץ `index.html` בדפדפן (Chrome מומלץ).
- גרור את הקובץ לחלון הדפדפן, או
- לחץ לחיצה כפולה על `index.html`, או  
- בטרמינל: `open index.html`

### צעד 2: הגדרות
- **בחר קטגוריה** מהרשימה הנפתחת (גיאוגרפיה, היסטוריה, מדע, וכו')
- **הזן שמות מתמודדים** או השאר את ברירת המחדל ("ימין" ו"שמאל")
- **הגדר זמן התחלתי** (ברירת מחדל: 45 שניות לכל מתמודד)
- **הגדר עונש דילוג** (ברירת מחדל: 3 שניות)
- **בחר מי מתחיל** - ימין או שמאל
- **לחץ "התחל משחק"**

### צעד 3: שחק!

#### מקשי שליטה:
- **SPACE** - תשובה נכונה (מעבר למתמודד השני)
- **S** - דילוג (הצג תשובה למשך 3 שניות)
- **ESC** - השהה משחק
- **R** - התחל מחדש

#### חשוב לדעת:
- כאשר לוחצים **S** (דילוג):
  - התשובה הנכונה מוצגת על המסך
  - השעון ממשיך לרוץ! (תראה 28→27→26→25)
  - אחרי 3 שניות, התשובה נעלמת והשאלה הבאה מופיעה
  - אותו מתמודד ממשיך (כבר ניכו לו 3 שניות)

- השעון הופך לאדום כשנשארות 10 שניות או פחות

- כשהשעון מגיע ל-0, המשחק נגמר והמתמודד השני מנצח

## 🎵 הוספת מוזיקה (אופציונלי)

1. הוסף קבצי MP3 לתיקייה `sounds/`:
   - `countdown.mp3` - ספירה לאחור (3, 2, 1, GO!)
   - `background.mp3` - מוזיקת רקע (תתנגן בלופ)
   - `correct.mp3` - צליל תשובה נכונה
   - `skip.mp3` - צליל דילוג
   - `gameover.mp3` - סיום משחק

2. המשחק יעבוד מצוין גם בלי אודיו!

## 📝 הוספת שאלות חדשות

1. פתח את הקובץ `js/questionsData.js`
2. הוסף שאלות לקטגוריות קיימות או צור קטגוריה חדשה:

```javascript
const QUESTIONS_DATA = {
  "קטגוריה חדשה": [
    {
      "type": "text",
      "content": "השאלה שלי?",
      "answer": "התשובה"
    }
  ]
};
```

## 🖼️ הוספת שאלות עם תמונות

1. שים תמונות בתיקייה `images/`
2. הוסף ל-`js/questionsData.js`:

```javascript
{
  "type": "image",
  "content": "images/my-picture.jpg",
  "answer": "התשובה לתמונה"
}
```

## 🎮 טיפים למשחק

1. **למסך מלא**: לחץ F11 בדפדפן
2. **חיבור למקרן**: חבר מחשב לטלוויזיה/מקרן והצג במסך מלא
3. **גודל טקסט**: השתמש ב-Ctrl+ או Ctrl- לכיוונון גודל
4. **משחק רציף**: השאלות מתחדשות אוטומטית כשנגמרות

## ❓ בעיות נפוצות

**השאלות לא נטענות?**
- בדוק שקובץ `questions.json` תקין
- ודא שהקובץ בקידוד UTF-8

**מוזיקה לא מתנגנת?**
- הוסף את הקבצים לתיקייה `sounds/`
- לחץ על המסך פעם אחת (דפדפנים חוסמים אודיו אוטומטי)

**המשחק לא מתחיל?**
- פתח את הקונסול (F12) ובדוק שגיאות
- נסה לפתוח את `index.html` מחדש
